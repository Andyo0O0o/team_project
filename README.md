# Star Delivery（星瀚餐递）

> 校园专属轻量外卖Web平台，无需下载APP，15-30分钟校内极速配送，让点餐更简单高效。

## 一、项目简介
### 核心定位
聚焦本校校园场景，解决校内点餐“沟通繁、时效慢、订单乱”痛点，提供“学生快速点餐+商家高效接单”的闭环服务。

### 核心亮点
- 🚀 校内极速配：配送范围≤1公里，时效15-30分钟，比校外外卖快50%
- 🖥️ 双端适配：响应式设计，电脑/手机浏览器直接使用，无需下载APP
- 🎯 操作极简：学生3步下单（选菜→填地址→支付），商家1步接单
- 🔐 校园适配：学号注册验真，支持微信/支付宝支付（后续可拓展校园卡）

## 二、核心功能
| 角色       | 核心功能列表                                                                 |
|------------|------------------------------------------------------------------------------|
| 学生用户   | 商家/菜品浏览、学号注册登录、购物车管理、订单提交支付、实时订单追踪、地址管理 |
| 商家用户   | 菜品管理（增删改查/批量操作）、实时订单提醒、接单/配送/完成操作、数据统计     |
| 管理员     | 商家入驻审核、订单监控、投诉处理、系统基础配置                               |

## 三、技术栈
| 类型       | 选型                                                                 |
|------------|----------------------------------------------------------------------|
| 前端技术   | Vue.js + Element UI + Bootstrap + Axios                               |
| 后端技术   | Java + Spring Boot + MyBatis + MySQL 8.0                             |
| 开发工具   | Git + GitHub + Postman + 墨刀（原型设计）                             |
| 环境依赖   | Node.js 14+、JDK 1.8+、MySQL 8.0、Maven 3.6+、Vue CLI 4+             |
